# Pulse Player

A responsive browser-based music player built for an internship project. It lets users load audio files from their device and play them in a polished playlist interface.

## Features

- Add multiple local audio files
- Play, pause, previous, and next controls
- Searchable playlist
- Shuffle and repeat modes
- Seek bar with current time and duration
- Volume control
- Animated visual artwork while music plays
- Responsive layout for desktop and mobile

## How to Use

Open `index.html` in a browser, choose audio files from your device, and start playing music.

## Files

- `index.html` - app structure
- `style.css` - visual design and responsive layout
- `script.js` - playlist and playback functionality
